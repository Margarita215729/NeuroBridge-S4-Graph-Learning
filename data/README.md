# Data directory

This repository does not store raw biomedical datasets by default.

Later phases will use processed outputs from the original NeuroBridge-S4 demonstration, such as:

- `pseudo_crew.csv`
- `deviation_scores.csv`
- `domain_scores.csv`
- `baci_scores.csv`
- `baci_sensitivity.csv`
- `crew_level_summary.csv`

Place local input files in `data/processed/` if needed. Do not commit large raw data caches.
